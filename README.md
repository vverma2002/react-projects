# react-projects
React Js Projects

**Setup**

**1. Install global create react app module**

*npm i -g create-react-app*

**2. Create a react app using the above module**

*cd react-projects*

*create-react-app test-app*

**3. Start dev mode**

*cd test-app*

*npm start*

**5. Add other modules**

*npm i bootstrap@4.1.1*
